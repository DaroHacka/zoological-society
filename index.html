Welcome to Your Game Archive!

This is your personal digital vault for organizing and exploring your video game collection.

Getting Started

  1. Add a Console: Click "Add console" in the top right to add your first game folder. You'll need to specify a name (e.g., "Nintendo Switch") and the absolute path to your ROMs folder.
  2. Scan for Games: Once a console is added, click "Rescan folder" to discover all games in that folder. The system will automatically detect game files based on their extension (.nsp, .xci, .iso, etc.).
  3. Fetch Metadata: Use "Fetch Text Metadata" to pull game descriptions from online databases. "Fetch Covers" and "Fetch Screenshots" will download box art and screenshots.
  4. Organize Your Collection: Mark games as favorites, track your progress (Playing, Completed, Plan to Play, Dropped, On Hold), and use filters to find specific games.

Features

  • Browse by Console: Games are organized by platform in the sidebar
  • Global Search: Search across your entire collection using the search bar above
  • Genre Filters: Filter games by genre within each console
  • Status Tracking: Track your gaming progress with status options
  • Visual Gallery: View box covers and screenshots for each game

Keyboard Shortcuts

  • ← → Navigate between games in detail view
  • Esc Close modals
  • Enter Perform search
