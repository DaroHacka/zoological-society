# Zoological Society - Initialize Script

## Quick Start

### Option 1: Python Script (Recommended)
```bash
python Zoological_Society.py
```

This will:
- Check if dependencies are installed
- Start backend on port 9001
- Start frontend on port 3021
- Display your local IP for network access
- Ask about auto-boot on first run

### Option 2: Bash Script
```bash
./initialize
```

This will:
- Start the backend (FastAPI) on port 9001
- Start the frontend (HTTP server) on port 3021
- Show live terminal output
- Auto-restart if either server crashes

## Access the Application

## Stop the Servers

Press **Ctrl+C** in the terminal running the script.

## Troubleshooting

### If ports are already in use:
```bash
pkill -f "python main.py"
pkill -f "http.server 3021"
```

### Or change ports in the script:
```bash
BACKEND_PORT=9002
FRONTEND_PORT=3022
```
